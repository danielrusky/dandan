Создание корневого каталога config

- подключаю виртальное окружение
- создаю readme, gitignore
- создаю django-приложение
- проверяем работоспособность
- db.sqlite3 появится
- добавляем requirements.txt
- загружаем библиотеки
- удаляю лишние библиотеки


Создание новый проект catalog

- создаю само приложение каталог
- настраиваю пару контроллеров
- настраиваю маршрутизацию
- настраиваю шаблоны
- подключение к базе данных
- engine,name,host
- user,password,port
- создаю папки static, media
- создаем модель в models.py
- прописываем в admin.py
